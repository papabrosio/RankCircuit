How I Built this Cyberpunk Blog for $0 using Gemini

Welcome to the machine. What you see before you is not a standard WordPress template or a bloated Squarespace site. It is a custom-engineered Liquid Neo-Glass environment built entirely through AI orchestration.

1. The Vision: Beyond the Matrix

The standard "falling green code" is a cliché. For Rank Circuit, I wanted something that felt more expensive, fluid, and modern. I prompted Gemini to help me design a "Liquid Data" aesthetic—where information flows organically across the screen rather than dropping in vertical lines.

2. The Prompt Engineering

The heart of this site is the Canvas animation. To achieve the horizontal fluid motion of binary streams, I used variations of this prompt:

"Write a high-performance Vanilla JS Canvas animation. Instead of a standard Matrix rain, create organic 'streams' of 0s and 1s that flow with a sine-wave offset. Use Phosphor Green (#00FF41) with varying opacities to create a sense of depth and liquid movement."

3. The Glassmorphism Layer

To make the content cards feel like frosted crystal, we leveraged deep backdrop blurs (blur(24px)) and a critical technical detail: The Noise Texture. By overlaying a very subtle SVG fractal noise, the glass gains a physical, grainy quality that breaks the sterile digital look of flat colors.

4. Deployment for Free

Infrastructure shouldn't cost you a dime in the prototype phase. We used GitHub Pages to host the entire platform. By pushing a single index.html (containing all logic and styles) to a public repository, the site is live globally in seconds.

5. Why Aesthetics Matter

In the age of agents, credibility is tied to the quality of your interface. A site that looks like it was built in 1998 suggests an algorithm that hasn't been updated since then. Rank Circuit is designed to signal that the data being ranked is as high-end as the frosted glass it sits on.

You can find the full source code for this architecture on my GitHub Repository.
